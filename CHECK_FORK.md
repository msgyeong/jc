# Fork 확인 방법

## 1. Fork 여부 확인

브라우저에서 접속:
```
https://github.com/k50004950-ctrl/jc
```

### 결과 A: 404 에러
→ **Fork 안 됨!** 아래로 이동

### 결과 B: 페이지 보임
→ **Fork 완료!** Railway 권한 설정으로 이동

---

## 2. Fork 하는 방법

1. https://github.com/msgyeong/jc 접속
2. 우측 상단 **"Fork"** 버튼 클릭
3. **"Create fork"** 클릭
4. 완료!

---

## 3. Railway 권한 설정

### Railway 화면에서:

1. **"Configure GitHub App"** 클릭
2. GitHub 설정 페이지 열림
3. 아래로 스크롤 → **"Repository access"**
4. **"Select repositories"** 선택
5. 드롭다운에서 **`jc`** 찾아서 선택 ✅
6. **"Save"** 버튼 클릭
7. Railway로 돌아가기
8. 🔄 새로고침

---

## 4. 다시 시도

Railway에서:
1. "Deploy from GitHub repo" 다시 클릭
2. 이제 `k50004950-ctrl/jc` 보일 거예요!
3. 클릭해서 배포 시작!

---

## 💡 여전히 안 보이면?

### Railway 화면 새로고침:
- 브라우저 새로고침: `F5` 또는 `Ctrl + R`
- Railway 페이지 다시 접속

### 또는:
- "Configure GitHub App" 옆 🔄 버튼 클릭
