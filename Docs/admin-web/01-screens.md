본 문서는 **관리자 웹**의 화면 목록과 화면 ID 규칙을 정의한다.

- 설계 기준: `../prd/08-design.md`
- 용어 기준: `../contributing.md`

---

## 1. 화면 ID 규칙

- 관리자 웹 화면 ID는 `AW-###` 형식을 사용한다.
- 하나의 화면은 하나의 주요 목적을 가진다.

---

## 2. 화면 목록(초안)

| 화면 ID | 화면명 | 목적 |
|---|---|---|
| AW-001 | 관리자 웹 로그인 | 최고 관리자 인증 |
| AW-002 | 가입 승인 목록 | 가입 대기/거절/승인 상태 관리 |
| AW-003 | 회원 목록 | 회원 조회/검색 |
| AW-004 | 회원 상세 | 회원 정보 조회/수정(관리자 전용 필드 포함) |
| AW-005 | 직책/권한 관리 | 직책별 권한 부여·회수 |
| AW-006 | 게시판 관리 | 게시글 목록 조회/삭제, 댓글 삭제 |
| AW-007 | 공지사항 관리 | 공지사항 작성/수정/삭제 |
| AW-008 | 일정 관리 | 일정 목록 조회, 등록/수정/삭제 |
| AW-009 | 관리자 계정 보안 | 비밀번호 생성/변경 등(정책 적용) |
| AW-010 | 대시보드(선택) | 통계/분석(추후 확장) |

---

## 3. 화면 간 기본 흐름

- AW-001 로그인 성공 → AW-002 가입 승인 목록
- 회원 조회가 필요한 경우 AW-003 → AW-004
- 권한/직책 설정이 필요한 경우 AW-005
- 게시판/공지/일정 운영이 필요한 경우 AW-006/007/008
