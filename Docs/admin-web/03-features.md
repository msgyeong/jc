# 관리자 웹 - 기능 정의

<!-- [변경일: 2026-01-19] 변경 이유: 관리자 웹 기능 정의 문서 신규 생성 -->
<!-- [변경일: 2026-01-19] 변경 이유: PRD(F-012/F-013) 기준으로 기능 범위 보완 -->

본 문서는 **관리자 웹**에서 제공해야 하는 기능을 정의한다.

- 설계 기준: `../prd/08-design.md`

---

## 1. 인증

- 최고 관리자 로그인
- 세션 유지/만료 정책(추후 확정)
- 접근 제어: 최고 관리자 인증 전에는 모든 관리자 기능 접근 불가

---

## 2. 회원 가입 승인

- 가입 대기 회원 목록 조회
- 승인/거절 처리
- 거절 사유 기록 및 사용자 안내(앱 로그인 시)

---

## 3. 회원 관리

- 회원 목록 조회/검색
- 회원 상세 조회
- 회원 상태 변경(정지/복구 등)
- 관리자 전용 필드 수정(허용 범위는 PRD 기준)

---

## 4. 직책/권한 관리

- 직책별 권한 부여·회수
- 앱 내 관리자(직책 기반) 권한 설정
  - 공지사항 작성 권한
  - 일정 등록 권한
  - (필요 시) 게시글 삭제 권한

---

## 5. 게시판/공지/일정 관리

- 게시판 관리
  - 게시글 목록 조회
  - 게시글 삭제
  - 댓글 삭제
- 공지사항 관리
  - 공지사항 작성/수정/삭제
  - 참석자 조사 활성화 옵션(공지 작성 시, PRD 정의 범위 내)
- 일정 관리
  - 일정 목록 조회
  - 일정 등록/수정/삭제

---

## 6. 관리자 계정 보안

- 관리자 웹에서 비밀번호 생성 및 변경을 통해 관리자 계정 보안을
  생성/유지/관리한다(`prd/04-non-functional.md` 기준).

---

## 7. 감사/추적(권장)

- 누가/언제/무엇을 변경했는지 로그를 남긴다(구현 범위는 추후 확정).
