# 영등포 JC 회원관리 앱
## 디자인 시스템 정의서 – 색상 스타일

본 문서는 영등포 JC 회원관리 앱의 디자인 시스템 중 색상(Color Palette)을
역할 기반으로 정의한 문서이다.

- 본 문서는 **스타일 기준 문서**이며 기능/화면/유저 플로우를 정의하지 않는다.
- 정의되지 않은 항목은 추측하지 않고 명시적으로 표시한다.

---

## 테마 정책

**중요**: 사용자 스마트폰의 '다크/라이트' 환경과 무관하게 앱은 **라이트 테마만 사용**한다.

- 다크 테마 팔레트는 문서에 포함될 수 있으나, **참고용**이며 실제 개발에서는 사용하지 않는다.

---

## 1. 색상 팔레트 정의

### 설계 원칙
- 커뮤니티 기반 앱 특성상 **신뢰감, 정돈됨, 공식성**을 우선
- 공지 / 일정 / 게시글 등 정보 위계가 명확히 드러나야 함
- 과도한 브랜드 컬러 사용 지양, **중립 색상 + 절제된 포인트**
- 라이트 테마 기준으로 가독성 우선

---

## 1-1. 라이트 테마 (Light Theme)

### 주 색상 (Primary Color)
- HEX: `#1F4FD8`
- 사용 목적:
  - 상단 AppBar
  - 주요 버튼(확인, 저장, 완료)
  - 핵심 인터랙션 강조
- 주는 인상:
  - 안정적이고 신뢰감 있는 블루
  - 공식 단체·조직형 서비스에 적합

### 보조 색상 (Secondary Color)
- HEX: `#E6ECFA`
- 사용 목적:
  - Primary 강조 보조 배경
  - 강조 박스/칩 배경

### 강조 색상 (Accent Color)
- HEX: `#F59E0B`
- 사용 목적:
  - 중요 알림/포인트 액션
  - 중요한 배지/태그

### 배경 색상 (Background)
- HEX: `#F9FAFB`

### 텍스트 색상
- 기본 텍스트: `#111827`
- 보조 텍스트: `#6B7280`

### 상태 색상
- 오류(Error): `#DC2626`
