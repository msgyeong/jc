# 영등포 JC 회원관리 앱
## 디자인 시스템 정의서 – 컴포넌트 스타일 정의

본 문서는 영등포 JC 회원관리 앱의 **컴포넌트 스타일 가이드**를 정의한다.

- 본 문서는 구현 코드가 아닌 **디자인 및 스타일 기준 문서**이다.
- 재사용 가능한 컴포넌트를 중심으로 작성한다.

---

## 1. App Bar

### 기본 형태
- 높이: 기본 모바일 App Bar 높이
- 좌측: 뒤로가기 아이콘 또는 메뉴 아이콘
- 중앙 또는 좌측: 화면 제목
- 우측: 필요 시 액션 아이콘

### 색상 사용 규칙
- 배경색: Primary Color
- 제목 텍스트: Text Primary
- 아이콘: 제목 텍스트와 동일 색상

### 상태 변화
- 기본 상태: Primary Color 배경
- 스크롤 연동 시:
  - PRD에 정의되지 않음
- 비활성 상태:
  - PRD에 정의되지 않음

### 사용 의도
- 화면의 맥락과 위치를 명확히 전달
- 커뮤니티 운영 앱의 공식적이고 안정적인 인상 제공
