## 1. 점검 요약

현재 앱은 기본 레이아웃과 CSS 변수는 갖추어져 있으나, **출시 수준의 완성도에는 미달**한다.
핵심 문제는 다음 세 가지로 요약된다:

| 등급 | 문제 | 영향 범위 |
|------|------|-----------|
| **심각** | 홈 배너 영역이 텅 빈 연보라 박스 | 홈 첫인상 |
| **심각** | N 배지가 텍스트 옆에 겹치며 레이아웃 깨짐 | 홈 일정 리스트 |
| **중간** | 일정 화면 캘린더가 렌더링 안 되고 텍스트만 표시 | 일정 탭 |
| **중간** | 하단 네비게이션 이모지 아이콘 — 플랫폼별 렌더링 불일치 | 전체 |
| **경미** | 프로필 정보 필드가 단순 텍스트 나열 | 프로필 탭 |
| **경미** | 게시글 카드 통계 영역이 밋밋함 | 게시판 탭 |

## 2. 화면별 상세 문제점

### 2-1. 홈 화면

| # | 문제 | 현재 상태 | 원인 |
|---|------|-----------|------|
| H1 | **배너 영역 빈 화면** | 연보라(#E6ECFA) 배경에 "영등포 JC" 텍스트만 표시, 높이 200px 빈 공간 | 배너 이미지 미등록 시 대체 UI 없음 |
| H2 | **N 배지 겹침** | 일정 제목 우측에 빨간 N 배지가 텍스트와 겹쳐서 잘림 | `.new-badge`가 inline으로 삽입되어 flex 정렬 없이 제목 옆에 붙음 |
| H3 | 공지사항 카드에 카드 border/shadow 없음 | 공지 제목과 시간만 텍스트로 나열 | 홈 요약 영역에 `.card` 클래스 미적용 |
| H4 | 일정 리스트 날짜 정보 없음 | 일정명과 장소만 표시, 날짜/시간 없음 | 홈 요약에서 날짜 렌더링 누락 |

### 2-2. 게시판 화면

| # | 문제 | 현재 상태 | 원인 |

## 3. CSS 일관성 문제

### 3-1. 하드코딩된 색상값

CSS에서 디자인 시스템 변수를 사용하지 않고 직접 HEX를 쓴 곳:

| 위치 | 하드코딩 값 | 올바른 변수 |
|------|-------------|-------------|
| `.btn-primary:hover` | `#1a42b8` | `--primary-dark` (미정의) |
| `.btn-secondary:hover` | `#d4dff7` | `--secondary-dark` (미정의) |
| `.cal-weekday.sat` | `#2563EB` | 토요일 색상 변수 미정의 |
| `.badge-admin` | `#8B5CF6` | 역할 배지 색상 변수 미정의 |
| `.badge-member` | `#3B82F6` | 역할 배지 색상 변수 미정의 |
| `.stat-card.stat-warning` | `#fff8e1`, `#ffe082` | 경고 색상 변수 미정의 |
| `.result-box h3` | `#2e7d32` | success 텍스트 변수 미정의 |
| `.btn-remove:hover` | `#b91c1c` | error-dark 변수 미정의 |

### 3-2. 중복 선언

| 선택자 | 문제 |
|--------|------|
| `.app-bar-action` | 라인 2082~2097과 라인 2250~2261에서 **두 번 선언**, 서로 다른 스타일 |
| `.btn-sm` | 라인 299~302와 라인 1538~1541에서 **두 번 선언** |
| `.content-section` | 라인 704~706과 라인 1488~1490에서 **두 번 선언** |

### 3-3. 미정의 확장 색상

디자인 시스템 변수(`:root`)에 없지만 필요한 색상:

| 필요 변수 | 용도 | 제안 값 |
|-----------|------|---------|
| `--primary-dark` | Primary hover | `#1a42b8` |
| `--primary-light` | 약한 Primary 배경 | `#EBF0FC` |
| `--success-color` | 성공 상태 (이미 CSS에 있으나 디자인 문서에 없음) | `#10B981` |
| `--success-light` | 성공 배경 | `#ECFDF5` |
| `--warning-color` | 경고 상태 | `#F59E0B` (accent와 동일) |
| `--warning-light` | 경고 배경 | `#FEF3C7` |
| `--error-light` | 에러 배경 | `#FEE2E2` |
| `--saturday-color` | 토요일 텍스트 | `#2563EB` |
| `--sunday-color` | 일요일 텍스트 | `#DC2626` (error와 동일) |

### 3-4. 하단 네비게이션 이모지 문제

현재 하단 네비게이션 아이콘이 **OS 이모지**를 직접 사용:
- `🏠` `📋` `📅` `👥` `👤`
- 플랫폼(iOS/Android/Windows)마다 렌더링이 달라 일관성 없음
- 크기 제어 불가 (font-size로만 조절, 정밀 제어 불가)
- active/inactive 색상 변경 불가 (이모지는 CSS color 무시)
