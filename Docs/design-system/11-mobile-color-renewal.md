## 1. 현재 문제 진단

### 1-1. 색상 혼란 맵

현재 앱에서 사용 중인 색상을 영역별로 정리하면:

| 영역 | 현재 색상 | 문제 |
|------|-----------|------|
| 배너 1 | `#1F4FD8` → `#3B82F6` (블루 그라디언트) | 괜찮음 |
| **배너 2** | **`#059669` → `#10B981`** (초록 그라디언트) | **전체 톤과 불일치** |
| **배너 3** | **`#7C3AED` → `#A78BFA`** (보라 그라디언트) | **전체 톤과 불일치** |
| 앱바 | `#1F4FD8` (CSS 변수) | Primary 변경 필요 |
| **아바타** | **8색 랜덤** (#1F4FD8, #059669, #F59E0B, #DC2626, #7C3AED, #EC4899, #0891B2, #EA580C) | **알록달록, 촌스러움의 주범** |
| 하단 네비 (active) | `#1F4FD8` (CSS 변수, 이모지라 적용 안됨) | 이모지 색상 제어 불가 |
| **관리자 배지** | **`#8B5CF6`** (보라) | **Primary 톤과 불일치** |
| 일정 카테고리 | 5색 (블루/그린/노랑/빨강/회색) | 과하게 다채로움 |
| 프로필 히어로 | `#1F4FD8` → `#3B82F6` 그라디언트 | 색상은 OK, Primary 통일만 필요 |

### 1-2. 핵심 문제 3가지

1. **색상이 너무 많다** — 앱 전체에 파랑/초록/보라/핑크/주황/노랑/빨강/시안이 동시에 등장
2. **통일된 브랜드 컬러가 부재** — 어디가 "우리 앱의 색"인지 불명확
3. **아바타 8색 랜덤**이 회원 목록/프로필/게시글 카드에 모두 등장하여 산만

## 2. 리뉴얼 컬러 팔레트

### 2-1. 핵심 원칙

> **"Blue + Gray" 2톤 베이스. 강조색은 최소한만.**

- **주조색(80%)**: `#2563EB` Blue + Gray 계열
- **보조색(15%)**: 상태 표시용 (성공 초록/위험 빨강/경고 노랑) — 기능적 용도만
- **강조색(5%)**: N배지 빨강 등 극히 제한적 포인트

## 4. N배지 사용 기준 정리

### 4-1. 현재 문제

N배지가 **모든 곳**에 붙어서 강조 효과가 없다:
- 홈 공지 목록: 모든 공지에 N
- 홈 일정 목록: 모든 일정에 N
- 게시판 카드: 모든 새 글에 N
- 하단 네비 아이콘 위: 탭별 N 표시

### 4-2. N배지 표시 규칙 (리뉴얼)

| 위치 | 표시 조건 | 숨김 조건 |
|------|-----------|-----------|
| **게시판 카드** | 작성 3일 이내 + **내가 안 읽은 글** | 읽은 글, 3일 경과 |
| **홈 공지 요약** | 작성 3일 이내 + **내가 안 읽은 공지** | 읽은 공지, 3일 경과 |
| **홈 일정 요약** | 등록 3일 이내 | 3일 경과 |
| **하단 네비** | 해당 탭에 **미읽음 콘텐츠 1개 이상** | 미읽음 0개 |

### 4-3. 최대 N배지 개수 제한

| 화면 | 최대 N배지 수 | 이유 |
|------|--------------|------|
| 홈 공지 목록 (3~5건) | **최대 2개** | 전부 N이면 강조 의미 없음 |
| 홈 일정 목록 (3~5건) | **최대 2개** | 동일 |
| 게시판 목록 (페이지당 20건) | **제한 없음** | 스크롤 가능하므로 OK |
| 하단 네비 | **탭당 dot 1개** | 숫자 대신 빨간 dot(6px)만 표시 |

### 4-4. 하단 네비 N배지 → Dot 전환

```
현재:
│  🏠  📋  📅  👥  👤  │   ← 각 탭에 [N] 텍스트 배지

변경 후:
│  🏠  📋•  📅  👥  👤  │   ← 미읽음 있는 탭에만 6px 빨간 dot
```

```css
.nav-item .nav-dot {
    width: 6px;
    height: 6px;
    border-radius: 50%;
    background: #DC2626;
    position: absolute;
    top: 4px;
    right: calc(50% - 16px);
}
```

## 6. 전체 색상 사용 빈도 가이드

각 색상이 앱에서 차지하는 비중을 명시한다:

| 비중 | 색상 | 사용처 |
|------|------|--------|
| **60%** | Gray 계열 (#F9FAFB, #FFFFFF, #F3F4F6, #D1D5DB) | 배경, 카드, 테두리, 구분선 |
| **25%** | Text 계열 (#111827, #6B7280, #9CA3AF) | 텍스트 |
| **10%** | Primary Blue (#2563EB, #DBEAFE, #EFF6FF) | 앱바, 버튼, 활성탭, 아바타 |
| **3%** | Danger Red (#DC2626, #FEE2E2) | N배지, 에러, 로그아웃 |
| **2%** | Success/Warning | 상태 배지 (활성/대기)만 |
