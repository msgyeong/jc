# 전화걸기 기능 기획서

> 작성일: 2026-03-10 | 작성자: 기획자 에이전트
> 관련 요구사항: R-08 (프로필 바로 통화)
> 우선순위: Must-have (MVP 범위)

---

## 1. 기능 개요

회원 목록 및 프로필에서 전화번호를 터치하면 바로 전화가 연결되는 기능.

- `tel:` 링크를 사용하여 모바일 기기의 기본 전화 앱 호출
- 별도 앱 설치 불필요 (브라우저 기본 기능 활용)
- 프론트엔드만으로 구현 가능 (백엔드 변경 없음)

---

## 2. 전화번호 포맷

| 항목 | 규칙 |
|------|------|
| **표시 형식** | `010-XXXX-XXXX` (하이픈 포함) |
| **tel: 링크** | `tel:010XXXXXXXX` (하이픈 제거) |
| **저장 형식 (DB)** | `01012345678` (숫자만, 기존 유지) |

### 포맷 변환 예시

```javascript
// DB → 표시
function formatPhone(phone) {
  if (!phone) return null;
  const digits = phone.replace(/\D/g, '');
  if (digits.length === 11) {
    return `${digits.slice(0,3)}-${digits.slice(3,7)}-${digits.slice(7)}`;
  }
  return phone;
}

// 표시 → tel: 링크
function toTelLink(phone) {
  if (!phone) return null;
  return `tel:${phone.replace(/\D/g, '')}`;
}
```

## 3. 적용 위치

### 3-1. 회원 목록 (`members.js`)

```
┌─────────────────────────────────┐
│  👤 김회장         회장          │
│  영등포건설(주)                  │
│  📞 010-1234-5678              │  ← 전화 아이콘 + 번호 클릭 시 전화
└─────────────────────────────────┘
```

- 회원 카드에 전화 아이콘(`📞`) + 번호 표시
- 번호 전체 영역이 클릭 가능 (`<a href="tel:...">`)
- 전화번호 없는 회원은 해당 행 미표시

### 3-2. 회원 상세 (회원 프로필 조회)

```
┌─────────────────────────────────┐
│  전화번호                       │
│  📞 010-1234-5678  [전화걸기]   │  ← 번호 클릭 또는 버튼 클릭
│                                 │
│  사업장 연락처                   │
│  📞 02-1234-5678   [전화걸기]   │  ← 사업용 번호도 동일 적용
└─────────────────────────────────┘
```

- 개인 전화번호 + 사업장 연락처 모두 `tel:` 링크 적용
- 전화 아이콘 버튼으로 시각적 강조

### 3-3. 내 프로필 (`profile.js`)

```
┌─────────────────────────────────┐
│  내 전화번호                    │
│  010-1234-5678                 │  ← 본인 프로필에서는 tel: 링크 미적용
└─────────────────────────────────┘
```

- 본인 프로필에서는 전화걸기 비활성 (본인에게 전화할 필요 없음)
- 전화번호 텍스트만 표시

---

## 4. 전화번호 없는 회원 처리

| 상황 | 처리 |
|------|------|
| `phone`이 NULL 또는 빈 문자열 | 전화 아이콘 + 번호 행 자체를 미표시 |
| 회원 목록 카드 | 전화 행 생략, 나머지 정보만 표시 |
| 회원 상세 | "전화번호 미등록" 텍스트 (회색, 비활성) |

## 5. UI/UX 세부사항

### 5-1. 전화 아이콘

- 아이콘: `📞` 또는 SVG phone 아이콘
- 색상: Primary `#2563EB`
- 크기: 20px (목록), 24px (상세)

### 5-2. 클릭 영역

```html
<!-- 회원 목록 -->
<a href="tel:01012345678" class="member-phone-link">
  <span class="phone-icon">📞</span>
  <span class="phone-number">010-1234-5678</span>
</a>

<!-- 회원 상세 -->
<a href="tel:01012345678" class="phone-call-btn">
  <span class="phone-icon">📞</span>
  전화걸기
</a>
```

## 6. 구현 범위

| 항목 | 담당 | 난이도 | 비고 |
|------|------|--------|------|
| 회원 목록 전화 링크 | 프론트엔드 | 낮음 | `members.js` 수정 |
| 회원 상세 전화 버튼 | 프론트엔드 | 낮음 | 회원 상세 뷰 수정 |
| 내 프로필 번호 표시 | 프론트엔드 | 낮음 | `profile.js` 수정 |
| 번호 포맷 유틸 함수 | 프론트엔드 | 낮음 | 공통 유틸에 추가 |
| CSS 스타일 | 프론트엔드 | 낮음 | `main.css` 추가 |

**백엔드 변경: 없음** (기존 API가 phone 필드를 이미 반환)

## 7. 데스크톱 대응

- 데스크톱 브라우저에서 `tel:` 클릭 시 Skype 등 VoIP 앱 연동 또는 "앱을 찾을 수 없음" 표시
- 모바일 우선 앱이므로 데스크톱 별도 대응 불필요
- 데스크톱에서는 번호 클릭 시 클립보드 복사로 대체하는 것을 향후 고려 가능
