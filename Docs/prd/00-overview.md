# PRD - 개요 및 프로젝트 범위

## 영등포 JC 회원관리 및 커뮤니케이션 통합 앱

---

## 1. 개요 (Overview)

### 1.1 프로젝트 배경

본 프로젝트는 한국 JCI 정회원 조직인 영등포 JC 로컬의 내부 운영과 회원 간 소통을 효율화하기 위해 기획된 회원관리 및 커뮤니케이션 통합 앱이다.

현재 영등포 JC의 운영 방식은 여러 플랫폼으로 분산되어 있다.

- 공식 홈페이지  
  유지·관리 부족, 정보 업데이트 지연
- 네이버 밴드  
  게시글 중심의 제한적 소통
- 카카오톡 단체 채팅방  
  실시간 대화 위주, 공지·일정·정보 검색에 구조적 한계

이로 인해 공지 전달의 일관성 저하, 일정 누락, 회원 정보의 비체계적 관리 문제가 지속적으로 발생하고 있다.

본 앱은 공지, 게시글, 일정, 회원 정보, 알림을 하나의 시스템으로 통합하는 것을 목표로 한다.

---

### 1.2 왜 이 앱이 필요한가

#### 소통 구조의 통합

현재 영등포 JC의 공지, 게시글, 일정, 소통은  
홈페이지, 네이버 밴드, 카카오톡 단체 채팅방으로 분산되어 있다.

이로 인해
- 공지가 어디에 올라왔는지 반복적으로 안내해야 하고
- 일정이 메시지 흐름에 묻혀 지나가며
- 과거 정보를 다시 찾기 어렵다.

본 앱은
- 공지
- 일반 게시글
- 일정
- 알림

을 **단일 앱에서 관리**함으로써  
"이 앱만 보면 된다"는 명확한 기준점을 제공한다.

---

#### 일정 관리 신뢰도 확보

기존 일정 관리는
- 게시글
- 카카오톡 메시지
- 개인 캘린더 입력

등으로 분리되어 있어 누락 가능성이 높다.

본 앱에서는
- 게시글 작성 시 일정 등록이 가능하고
- 등록된 일정은 캘린더에 자동 반영된다.

또한
- 일정 기준 알림 구조를 사전에 설계하여
- MVP 이후 일정 리마인더 푸시 알림을 활성화할 수 있는 기반을 마련한다.

이를 통해
- "일정을 몰랐다"
- "까먹었다"
와 같은 상황을 구조적으로 줄인다.

---

#### 회원 네트워크 강화

본 앱은 단순한 공지 앱이 아니라  
**회원 간 관계를 확장하고 연결하는 네트워크 플랫폼**을 지향한다.

회원 가입 시 다음 정책을 전제로 한다.

- 회원 가입 시 개인정보 활용방안에 대한 동의를 얻는다.
- 회원 가입 화면에서 수집하는 정보는 관리자 웹에서 확인 및 보완할 수 있다.
- 프로필 정보 공개 항목
  - 필수 공개 항목
    - 이름(성명)
    - 프로필 사진 (회원가입 시 1매 업로드는 필수일 수 있음)
    - 소속 JC
    - JC 직책
  - 선택 항목 (공개/비공개 설정 가능)
    - 연락처
    - 직장명
    - 직장 직책 또는 직위
    - 업종
    - 거주지 지역
    - 사업지 또는 근무지 지역
    - 직장 주소
    - 거주지 주소
    - 학력/경력
    - 가족 정보
    - 취미/특기
    - 추천인
- 민감정보(예: 주민등록번호)는 기본 비공개이며 필요 시 관리자 웹에서만 열람한다.
- 개인정보는 공개/비공개 설정을 할 수 있다.

이를 통해
- 단순히 "같은 단체 회원"이 아닌
- 서로의 역할, 직업, 배경을 이해한 상태에서
- 협력 가능한 관계를 형성할 수 있도록 한다.

---

#### 검색 기반 관계 확장

이 앱의 핵심 가치는 **속성 기반 회원 검색**이다.

기존 메신저나 연락처 기반 관리에서는
- 이름을 모르면 찾기 어렵고
- 직업이나 역할 중심의 탐색이 불가능하다.

본 앱에서는 다음과 같은 검색을 제공한다.

- 업종 검색  
  예: 변호사, 회계사, 요식업, 임대업, 보험 등
- 지역 검색  
  예: 강남, 강남구, 마포구, 신길동, 영등포, 영등포구, 영등포동 등
- 조직 직책 검색  
  예: 회장, 상임부회장, 이사, 회원관리이사, 회원, 관리자
- 교육 기수 검색  
  예: 신입회원교육 기수, 리더십 교육 기수

이를 통해
- 수직적 관계(회장단, 임원, 회원)
- 수평적 관계(동기, 업종, 지역)

가 동시에 연결되는  
**거미줄 형태의 회원 네트워크 구조**를 형성한다.

---

### 1.3 운영 정책 요약 (회원가입 / 관리자 / 개인정보)

#### 회원가입 완료 조건

- 회원가입 시 작성한 내용은 **관리자 웹**에서 확인한다.
- **관리자 웹에서 가입 승인을 해야 회원가입이 완료**된다.

#### 관리자 웹의 역할

- 회원가입 **승인** 및 **승인 중지**(탈퇴, 정지 회원 등)
- 회원 정보 보완 입력(예: 회원들의 직책 입력/수정)

#### 관리자 체계

- **최고 관리자**: 관리자 웹
- **관리자 계정**: 관리자 웹으로부터 권한을 부여받은 특정 회원

#### 앱 내 관리자 역할(직책 기반)

- 특정 직책(회장, 상임부회장, 내무부회장, 사무국장)은 앱에서 관리자 역할을 수행할 수 있다.
- 관리자 역할 범위(앱):
  - 공지 게시글 작성
  - 일정 등록
  - 모든 게시글 및 댓글 삭제

#### 개인정보 동의 및 공개 정책

- 회원가입 시 **개인정보 활용방안에 대한 동의**를 받는다.
- **필수 공개 항목**: 이름 / 소속 JC / JC 직책
- 그 외 개인정보는 **공개/비공개 설정**을 할 수 있다.

## 2. 프로젝트 범위 (Scope)

### 2.1 포함 범위 (In Scope)

- 회원 가입 및 로그인(관리자 승인 기반)
- 회원 프로필 관리
  - 필수 공개 항목: 이름 / 소속 JC / JC 직책
  - 선택 항목 공개/비공개 설정
- 공지 게시판 및 일반 게시판
- 게시글 기반 일정 등록
- 캘린더 기능
- 푸시 알림(기반 구조)
- 회원 검색 기능
- 관리자 웹 개발
  - 회원가입 승인 및 승인 중지(탈퇴, 정지 회원 등)
  - 회원 정보 보완 입력(예: 직책 입력/수정)
  - 관리자 계정 지정(최고 관리자: 관리자 웹 → 관리자 계정: 권한 부여된 특정 회원)
  - 직책 기반 앱 관리자 권한 부여(회장, 상임부회장, 내무부회장, 사무국장)

### 2.2 제외 범위 (Out of Scope)

- 회비 결제 기능
- 통계 및 분석 대시보드
- 실시간 채팅 기능

---

## 3. 앱 소개 (Product Definition)

### 3.1 앱 이름
- 영등포 JC

초기 버전은 영등포 JC 로컬 전용으로 운영한다.  
향후 한국 JCI 전체를 대상으로 한 차기 버전 앱을 별도 출시할 수 있으며, 기존 영등포 JC 앱은 유지한다.

### 3.2 핵심 가치 제안
- 회원들의 커뮤니티 활성화 및 교류를 통한 회원 간 성장 도모

### 3.3 제품 목적
- 영등포 JC 회원 간의 공지 전달, 소통, 일정 공유, 회원 정보 관리를 하나의 앱에서 제공한다.
- 오프라인 중심 커뮤니티 운영을 온라인으로 보조한다.
