### 페르소나 1. 핵심 사용자 (회원관리이사 / 관리자 웹 관점)

#### 기본 정보
- 이름: 경민수
- 나이: 42세
- 소속: 영등포 JC
- 영등포 JC 직책: 회원관리이사
- 직장명: 프로티나
- 직장 직책: 수석연구원
- IT 숙련도: 중간
- 사용 기기: PC(관리자 웹), Android 스마트폰

#### 목표 (Goals)
- 회원 정보(이름, 소속 JC, JC 직책, 연락처, 직업, 직장명, 직장 직책 또는 직위 등)를 체계적으로 관리하고 싶다.
- 회원 정보를 쉽게 검색하고, 이를 통해 회원 간 성장을 도모하고 싶다.
- 회원 가입 시 받은 정보와 앱/관리자 웹에서 추가로 입력된 정보를 취합하여,
  회원별 "회원 관리 카드"를 구성하고 한 화면에서 모든 회원 정보를 확인하고 싶다.
- 공지와 일정을 한 번에 회원들에게 전달하여 누락을 최소화하고 싶다.
- 회원 간 교류를 활성화하여 조직의 결속력을 높이고 싶다.
- 회원들의 교육 이력(신입회원교육, 2차 교육, 리더십 교육 등)을 체계적으로 관리하고,
  각 회원의 프로필에서 확인할 수 있도록 하고 싶다.
- 연수/교육 현황(예: 연수 이수/미이수)을 기준으로 회원을 필터링/정렬하여,
  대상자를 빠르게 확인하고 싶다.
- 프로필 사진과 입력된 정보를 통해 "누가 어떤 사람인지"를 빠르게 파악하고 싶다.

#### 기존 문제점 (Pain Points)
- 공지 및 일정이 여러 채널(홈페이지, 밴드, 카카오톡)로 흩어져 있다.
- 중요한 정보가 카카오톡 메시지에 묻혀 사라진다.
- 회원 직업, 연락처, 배경을 일일이 물어봐야 한다.
- 신규 회원이 조직과 사람을 이해하는 데 시간이 오래 걸린다.

#### 앱을 쓰는 이유 (Motivation)
- 공지, 일정, 회원 관리를 하나의 앱에서 처리하기 위해서
- 앱 내 관리자(직책 기반) 권한으로 공지 게시글과 일정 등록을 간편하게 하기 위해서
- Push 알림을 통해 공지와 일정을 빠르게 전달하고 리마인드하기 위해서
- 관리자 웹에서 회원 검색 후, 선택한 회원의 "회원 관리 카드"로 모든 정보를
  즉시 확인하기 위해서
- 연수/교육 현황 등을 기준으로 회원을 빠르게 필터링/정렬하여 확인하기 위해서

### 페르소나 2. 주요 사용자 (현역 회원)

#### 기본 정보
- 이름: 김지훈
- 나이: 36세
- 소속: 영등포 JC
- 영등포 JC 직책: 일반 회원
- 직장명: 삼성스마트앱개발연구소
- 직장 직책: 대표이사

- 사용 기기: iPhone

#### 목표 (Goals)
- 중요한 공지를 놓치지 않고 확인하고 싶다.
- 캘린더를 통해 영등포 JC 행사 일정을 한눈에 확인하고 싶다.
- 다른 회원들의 JC 직책, 직업, 연락처를 쉽게 알고 싶다.
- 필요할 때 도움을 받을 수 있는 회원을 빠르게 찾고 싶다.
- 회원들의 입회 년도를 파악하여 선배, 동기, 후배 관계를 이해하고 싶다.

- 카카오톡 단체방이 너무 많이 존재한다
  (전체 회원방, 현역방, 위원회방, 동기방 등).
- 단체방에서 메시지가 너무 많이 쌓여 중요한 정보를 놓친다.
- 공지가 전화로 전달되는 경우, 전화를 받지 못하면 정보를 놓친다.
- 회원 목록이 없어 카카오톡 프로필에 의존해야 한다.
- 이름과 얼굴을 매칭하기 어렵고, 어떤 일을 하는지 알기 어렵다.
- 행사 일정을 개인 캘린더에 직접 입력해야 하는 번거로움이 있다.

#### 앱을 쓰는 이유 (Motivation)
- 공지와 일정이 정리된 형태로 제공되기 때문에
- Push 알림을 통해 중요한 일정을 놓치지 않기 위해
- 회원 검색을 통해 네트워크를 확장하기 위해

- 조직 운영을 효율적으로 할 수 있기 때문에
- 회원과의 소통 부담을 줄이기 위해

## 사용자 정의

### 회원
- 모든 회원은 동일한 사용자 경험을 가진다.
- 특우회 여부는 **프로필 정보로만 구분**된다.
- 기능 접근 권한의 기준이 아니다.

### 최고 관리자
- `manager_###` 계정으로 **관리자 웹**에서 접근하는 관리자
- 가입 승인/회원 관리/직책 및 권한 부여·회수 수행

### 관리자 계정
- 최고 관리자가 **관리자 웹**에서 권한을 부여한 관리자 계정
- 최고 관리자의 업무를 위임받아 일부 관리 기능을 수행

### 앱 내 관리자(직책 기반)
- 최고 관리자가 **관리자 웹**에서 특정 회원(직책)에 권한을 부여하여,
  **앱**에서 관리자 기능을 수행하는 회원
- 공지사항 작성, 일정 등록 등
