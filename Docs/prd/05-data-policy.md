# PRD - 데이터 및 정책 개요

## 9. 데이터 및 정책 개요

### 주요 데이터
- 회원
- 게시글
- 일정
- 알림

### 데이터 정책
- 회원 탈퇴 시 계정 비활성
- 게시글 및 댓글 유지
- 관리자 삭제 시 게시글 비노출

### 접근 정책
- **가입 승인된 회원만 앱 사용 가능**
- **로그인하지 않은 사용자는 앱의 모든 기능에 접근 불가**
  - 로그인하지 않은 상태에서는 모든 화면 접근 차단
  - 하단 탭 네비게이션 숨김
  - 로그인 화면으로 자동 리다이렉트
- 로그인한 사용자만 접근 가능
- 선택 항목은 입력된 경우에만 노출

### Storage 정책 (파일 저장소)

#### 프로필 사진
- **저장 위치**: Supabase Storage `profiles` 버킷
- **경로 규칙**: `profiles/{auth_user_id}/avatar.jpg`
- **파일 제한**: 500MB 이하, 이미지 파일만 (jpeg, png, webp)
- **접근 정책**:
  - **조회**: 로그인한 회원은 모든 회원의 프로필 사진 조회 가능
  - **업로드/수정/삭제**: 본인 폴더(`profiles/{자기ID}/`)에만 허용
- **회원가입 시 업로드 순서**: Auth 계정 생성 → 프로필 사진 업로드 → 회원 정보 저장

#### 게시글 첨부 이미지 (추후)
- 게시글당 최대 5개
- 각 파일 500MB 이하
