# PRD - 릴리즈 기준

## 11. 릴리즈 기준

### MVP 기준
- 회원 가입 및 로그인 (가입 승인 절차 포함)
- 회원 프로필 관리
- 게시판(공지 게시판 / 일반 게시판)
- 일정 및 캘린더
- 회원 검색
- 관리자 계정 운영
- 관리자 웹 (회원 승인 기능 포함)
- Push 알림 기능 비활성 허용

### MVP 범위 재확인 (v3.0)

#### MVP 포함
- 홈(광고/배너, 공지 게시글 5개, 다가오는 일정 5개)
- 게시판
  - 공지 게시판(목록/상세/작성)
  - 일반 게시판(목록/상세/작성)
- 일정(캘린더/상세/등록)
- 회원(목록/프로필)
- 공통 기능(게시글/공지/일정): 댓글, 대댓글, 공감

#### MVP 제외(확장)
- 알림 센터 고도화
- 홍보/광고 상세
- 수익 모델 관련 기능

### v1.0 기준
- 승인 완료 Push 알림 활성화
- 일정 리마인더 Push 알림 활성화
- 실사용 테스트 완료
- 성능 및 안정성 기준 충족

### 출시 불가 조건
- 승인 없이 관리자 기능 사용 가능
- 개인정보 보호 정책 위반
- 공지 및 일정 신뢰성 붕괴
