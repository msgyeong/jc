## 📝 참고 사항

- **데모 모드**: `config.js`에서 `DEMO_MODE: true`로 설정하면 Supabase 없이 샘플 데이터로 테스트 가능
- **Supabase 연동**: `SUPABASE_URL`과 `SUPABASE_ANON_KEY` 설정 필요
- **권한 관리**: 공지 작성, 일정 등록 권한은 `role_permissions` 테이블에서 관리
- **프로필 이미지**: Supabase Storage의 `profiles` 버킷 사용
