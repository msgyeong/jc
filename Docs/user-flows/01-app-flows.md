## UF-001 앱 진입 및 홈 화면 진입

### 플로우 개요
사용자가 앱을 실행하고 인증 후 홈 화면에 진입하는 과정

### 진입 경로
스플래시 → 로그인(필요 시) → 홈

### 관련 화면
- S-001 (홈)
- S-011 (내 프로필)

### 관련 기능
- F-AUTH (인증 시스템)
- F-001 (홈 화면 구성 및 요약 정보 제공)
- F-011 (프로필 상세 조회 및 내 프로필 관리)

### 단계별 흐름
1. **스플래시 화면**
   - 앱 실행 시 `assets/images/jc_logo_slash.png` 이미지 표시
   - 서버 데이터 로딩 (최대 3초)
   - 인증 상태 확인

2. **로그인 (필요 시)**
   - 미인증 사용자: 로그인 화면으로 이동
   - 가입 승인된 회원만 로그인 가능
   - 자동 로그인 유지 기능 지원

3. **홈 화면 진입**
   - 공지사항 요약 (2~5개)
   - 다가오는 일정 요약 (2~5개)
   - 홍보/광고 슬라이드

### 비정상 흐름
- 가입대기 상태: 승인 진행중 화면으로 리다이렉트
- 거절 상태: 로그인 차단, 거절 사유 안내
- 정지 상태: 로그인 차단, 정지 안내
- 탈퇴 상태: 로그인 차단

## UF-002 홈 화면에서 주요 콘텐츠 탐색

### 플로우 개요
사용자가 홈 화면에서 게시글, 공지사항, 일정을 탐색하고 상세 내용을 확인하는 과정

### 진입 경로
홈 → 게시글/공지/일정 목록 → 상세 화면

### 관련 화면
- S-001 (홈)
- S-002 (게시글 목록)
- S-003 (게시글 상세)
- S-005 (공지사항 목록)
- S-006 (공지사항 상세)
- S-007 (일정 목록 / 캘린더)
- S-008 (일정 상세)

### 관련 기능
- F-001 (홈 화면 구성 및 요약 정보 제공)
- F-002 (게시글 목록 탐색)
- F-003 (게시글 상세 조회)
- F-005 (공지사항 목록 및 상세 조회)
- F-007 (일정 목록 및 캘린더 조회)
- F-008 (일정 상세 조회)
- F-015 (참석자 조사 기능)

### 단계별 흐름

#### 게시글 탐색 흐름
1. 홈 → 게시글 요약 카드 선택 또는 "게시판" 탭 선택
2. 게시글 목록 화면 진입 (최신순, 무한 스크롤)
3. 게시글 선택
4. 게시글 상세 화면 진입 (본문, 첨부 이미지, 댓글/대댓글, 공감)
5. 뒤로 이동 → 게시글 목록 복귀 (스크롤 위치 유지)

#### 공지사항 탐색 흐름
1. 홈 → 공지사항 요약 카드 선택 또는 공지사항 목록 진입
2. 공지사항 목록 화면 진입 (고정 공지 우선 노출)
3. 공지 선택
4. 공지 상세 화면 진입 (본문, 참석자 조사, 댓글/대댓글, 공감)
5. 뒤로 이동 → 공지 목록 복귀

#### 일정 탐색 흐름
1. 홈 → 일정 요약 카드 선택 또는 일정 목록/캘린더 진입
2. 일정 목록 또는 캘린더 화면 진입 (날짜 기준 정렬)
3. 일정 선택
4. 일정 상세 화면 진입 (날짜, 시간, 장소, 설명, 댓글/대댓글, 공감)
5. 뒤로 이동 → 일정 목록/캘린더 복귀

### 읽음 추적
- 상세 화면 진입 시 해당 콘텐츠의 읽음 상태 기록
- 'N' 배지 표시 조건: 생성일 3일 이내 + 미읽음 + (일정의 경우) 시작일 미경과

## UF-003 게시글 작성

### 플로우 개요
사용자가 새로운 게시글을 작성하여 공유하는 과정

### 진입 경로
게시글 목록 → 게시글 작성 화면

### 관련 화면
- S-002 (게시글 목록)
- S-004 (게시글 작성)

### 관련 기능
- F-002 (게시글 목록 탐색)
- F-004 (게시글 작성)

### 단계별 흐름
1. 게시글 목록에서 "작성" 버튼 선택
2. 게시글 작성 화면 진입
3. 제목 입력 (필수)
4. 본문 입력 (필수)
5. 이미지 첨부 (선택, 최대 5개까지, Supabase Storage 업로드)
6. 작성 완료 버튼 선택
7. 유효성 검사 (필수 입력 항목 확인)
8. 게시글 저장
9. 게시글 목록 또는 상세 화면으로 이동

### 비정상 흐름
- 필수 입력 미완료: 작성 완료 불가, 해당 항목 하단에 오류 메시지 표시
- 네트워크 오류: 재시도 버튼 제공, 입력값 유지

## UF-004 공지 게시글 작성

### 플로우 개요
**앱 내 관리자(직책 기반)**가 공지사항을 작성하는 과정

### 진입 경로
공지사항 목록 → 공지사항 작성 화면

### 관련 화면
- S-012 (공지사항 작성)

### 관련 기능
- F-006 (공지사항 작성)
- F-015 (참석자 조사 기능)

### 단계별 흐름
1. 권한 확인 (앱 내 관리자(직책 기반)만 접근 가능)
2. 공지사항 작성 화면 진입
3. 제목 입력 (필수)
4. 내용 입력 (필수)
5. 참석자 조사 활성화 옵션 선택 (선택)
6. 작성 완료 버튼 선택
7. 유효성 검사
8. 공지사항 저장
9. 공지사항 목록 및 홈에 즉시 반영

### 비정상 흐름
- 권한 없음: 접근 차단 또는 에러 메시지 표시
- 필수 입력 미완료: 작성 완료 불가

## UF-007 회원 검색 및 프로필 조회

### 플로우 개요
사용자가 다른 회원을 검색하고 프로필 정보를 확인하는 과정

### 진입 경로
하단 탭 → 회원 목록 → 프로필 상세

### 관련 화면
- S-009 (회원 목록 / 검색)
- S-010 (프로필 상세)

### 관련 기능
- F-010 (회원 목록 및 검색)
- F-011 (프로필 상세 조회 및 내 프로필 관리)

### 단계별 흐름
1. 하단 탭에서 "회원" 선택
2. 회원 목록 화면 진입
3. 검색어 입력 (이름, 직업, JC 직책, 소속 JC, 지역, 교육 기수 등)
4. 검색 결과 목록 표시
5. 회원 선택
6. 프로필 상세 화면 진입 (공개 정보 표시)
7. 뒤로 이동 → 회원 목록 복귀

### 프로필 정보 표시
- 관리자가 입력한 항목: 자동 공개 (소속 JC, JC 직책, 가입일, 연수, 표창, 국제대회 참가, JC경력)
- 관리자가 입력하지 않은 항목: 비공개
- 특우회 여부: 정보 표시용 (기능 차별 없음)

## 유저 플로우 간 관계

### 플로우 의존성
