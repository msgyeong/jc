# 홈 → 게시글 목록 → 게시글 상세
통합 유저 플로우 설계

<!-- [변경일: 2026-01-19] 변경 이유: 버전 표기 제거, 용어 통일(contributing.md 기준) -->

본 문서는 홈 화면에서 시작하여 게시글 목록을 거쳐 게시글 상세로 이동하는 **하나의 연속된 핵심 사용자 흐름**을 묶어서 정의한다.

---

## 통합 플로우 개요

- 플로우 목적  
  사용자가 홈 화면에서 게시글 콘텐츠를 인지하고,
  게시글 목록을 탐색한 뒤,
  특정 게시글의 상세 내용을 확인한다.

- 전체 흐름 (한 줄 요약)  
  홈 진입 → 게시글 요약 또는 게시판 진입 → 게시글 목록 탐색 → 게시글 선택 → 게시글 상세 확인

- 대상 사용자  
  가입 승인된 회원 (일반 회원 및 앱 내 관리자(직책 기반))

---

## 단계별 상세 유저 플로우

### Step 1. 홈 화면 진입

- 화면명  
  홈

- 사용자 행동  
  - 앱 실행
  - 또는 하단 탭에서 홈 선택

- 시스템 동작  
  - 홈 화면 구성 로딩
  - 공지사항 요약 영역 표시
  - 다가오는 일정 요약 영역 표시
  - (존재 시) 게시글 요약 카드 또는 게시판 진입 버튼 표시

---

### Step 2. 홈 화면에서 게시글 영역 인지

- 화면명  
  홈

- 사용자 행동  
  - 게시글 요약 카드 선택
  - 또는 "게시판" 탭 선택
  - 또는 "전체 게시글 보기" 버튼 선택

- 시스템 동작  
  - 게시글 목록 화면으로 이동 준비

---

### Step 3. 게시글 목록 화면 진입

- 화면명  
  게시글 목록

- 사용자 행동  
  - 게시글 목록 화면 진입

- 시스템 동작  
  - 게시글 목록 데이터 로딩
  - 최신 게시글 기준으로 리스트 표시

---

### Step 4. 게시글 목록 탐색

- 화면명  
  게시글 목록

- 사용자 행동  
  - 스크롤을 통해 게시글 탐색
  - (존재 시) 검색어 입력 또는 필터 선택

- 시스템 동작  
  - 추가 게시글 로딩 : 무한 스크롤 (초기 50개 노출)
  - 검색/필터 조건에 맞는 목록 갱신

---

### Step 5. 게시글 선택

- 화면명  
  게시글 목록

- 사용자 행동  
  - 특정 게시글 카드 선택

- 시스템 동작  
  - 선택된 게시글의 상세 화면으로 이동

---

### Step 6. 게시글 상세 화면 진입

- 화면명  
  게시글 상세

- 사용자 행동  
  - 게시글 상세 내용 확인

- 시스템 동작  
  - 게시글 제목, 작성자, 작성일, 본문, 첨부 이미지 로딩
  - (권한 시) 수정/삭제 메뉴 표시
  - 댓글/대댓글/공감 영역 표시

---

### Step 7. 게시글 상세 이후 분기

- 화면명  
  게시글 상세

- 사용자 행동  
  - 뒤로 버튼 선택

- 시스템 동작  
  - 게시글 목록 화면으로 복귀
  - 목록 스크롤 위치 및 상태 유지

---

## 통합 유저 플로우 표

| Step | 화면명 | UI 구성 | 사용자 행동 | 시스템 동작 |
|---:|---|---|---|---|
| 1 | 홈 | 홈 전체 영역 | 앱 실행/홈 진입 | 홈 콘텐츠 로딩 |
| 2 | 홈 | 게시글 요약/게시판 진입 | 게시글 영역 선택 | 게시글 목록 이동 |
| 3 | 게시글 목록 | 게시글 리스트 | 화면 진입 | 게시글 목록 로딩 |
| 4 | 게시글 목록 | 리스트/검색 | 탐색/검색 | 목록 갱신 |
| 5 | 게시글 목록 | 게시글 카드 | 게시글 선택 | 게시글 상세 이동 |
| 6 | 게시글 상세 | 본문 영역 | 상세 확인 | 게시글 데이터 표시 |
| 7 | 게시글 상세 | 뒤로 버튼 | 뒤로 이동 | 게시글 목록 복귀 |

---

## 플로우 설계 검증 포인트

- 홈에서 게시글로 진입하는 경로가 최소 1개 이상 존재한다.
- 게시글 목록은 탐색 전용이며, 상세 소비는 상세 화면에서만 이루어진다.
- 게시글 상세에서 뒤로 이동 시 사용자는 탐색 맥락을 잃지 않는다.
- 홈 → 목록 → 상세 흐름은 다른 콘텐츠(공지, 일정)와 동일한 패턴을 유지한다.
