# 게시글 작성 화면 텍스트 와이어프레임

<!-- [변경일: 2026-01-19] 변경 이유: 버전 표기 제거, 댓글/대댓글/공감 기능 포함 -->

---

## [게시글 작성 화면]

### 화면 목적
- 사용자가 새로운 게시글을 작성하여 공유한다.
- 필수 정보 입력을 명확히 안내한다.
- 작성 중 이탈 또는 완료 흐름을 명확히 제공한다.

---

## 상단 영역

### 구성 요소
- 취소 버튼
- 화면 제목: 게시글 작성
- 작성 완료 버튼

### 위치
- 화면 상단 고정 영역

### 역할
- 작성 흐름 제어
- 작성 완료 또는 작성 취소 진입 제공

---

## 중앙 영역 (스크롤 영역)

### 제목 입력 영역

#### 구성 요소
- 제목 입력 필드
  - 단일 줄 텍스트 입력

#### 위치
- 중앙 영역 상단

#### 역할
- 게시글 핵심 요약 정보 입력

---

### 본문 입력 영역

#### 구성 요소
- 본문 입력 필드
  - 여러 줄 텍스트 입력
  - 줄바꿈 허용

#### 위치
- 제목 입력 영역 하단

#### 역할
- 게시글 주요 내용 입력

---

### 이미지 첨부 영역

#### 구성 요소
- 이미지 첨부 버튼 (최대 5개까지, 네이버 밴드 방식)
- 첨부된 이미지 미리보기 리스트 (그리드 레이아웃)
- 각 이미지별 삭제 버튼

#### 위치
- 본문 입력 영역 하단

#### 역할
- 게시글 보조 콘텐츠 첨부 (최대 5개)
- 첨부 이미지 확인 및 제거
- 이미지 다중 선택 지원

---

## 하단 영역

### 구성 요소
- 입력 보조 영역
  - 키보드 상태에 따라 자동 조정

### 위치
- 화면 하단

### 역할
- 입력 흐름 방해 최소화
- 모바일 입력 환경 대응

---

## 게시글 작성 화면 유저 플로우 표

| Step | 화면명 | UI 구성 | 사용자 행동 | 시스템 동작 |
|---:|---|---|---|---|
| 1 | 게시글 작성 | 제목 입력 | 제목 입력 | 입력값 반영 |
| 2 | 게시글 작성 | 본문 입력 | 본문 입력 | 입력값 반영 |
| 3 | 게시글 작성 | 이미지 첨부 | 이미지 선택 (최대 5개) | 미리보기 표시 (그리드) |
| 4 | 게시글 작성 | 작성 완료 | 완료 선택 | 필수 입력 검증 |
| 5 | 게시글 작성 | 결과 처리 | 대기 | 게시글 등록 |
| 6 | 게시글 상세 | 게시글 화면 | 등록 완료 | 상세 화면 이동 |

---

## 설계 메모

- 제목과 본문은 필수 입력이다.
- 필수 항목 누락 시 작성 완료는 불가하다.
- 작성 취소 시 이탈 경고 여부는 PRD 기준을 따른다.
- 게시글 등록 성공 후 이동 화면은 PRD 기준을 따른다(상세 또는 목록).
- 게시글 작성 완료 후, 해당 게시글에 댓글/대댓글/공감 기능이 제공된다.
