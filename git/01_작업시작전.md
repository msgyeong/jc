# 🟢 작업 시작 전 루틴 (minsu 브랜치 기준)

아래 내용을 그대로 실행해줘.

1. 현재 브랜치가 `minsu`인지 확인해줘.
   - `minsu`가 아니면 `git checkout minsu` 해줘.

2. `origin/main` 최신을 가져오기 위해 `git fetch origin` 실행해줘.

3. 내 브랜치(`minsu`)에 `origin/main`을 rebase로 반영해줘.
   - 명령어: `git rebase origin/main`
   - rebase가 안 되는 상황이면 merge로 대신 해줘.

4. 충돌이 발생하면:
   - 충돌 난 파일 목록을 알려줘.
   - 각 파일에 대해 "내 변경 유지 / main 변경 채택 / 직접 수정" 중 어떻게 할지 선택지를 줘.

5. 마지막에 `git status` 결과를 요약해줘.
