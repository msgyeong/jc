# 🟢 작업 끝난 후 루틴 (minsu 브랜치 기준)

아래 내용을 그대로 실행해줘.

1. `git status`를 실행하고, 변경된 파일 목록을 보여줘.

2. 변경사항 전체를 `git add .` 으로 스테이징해줘.

3. 커밋 메시지를 "작업내용: (한줄요약)" 형태로 변경 내용에 맞게 추천해줘.
   - 내가 승인하면 그 메시지로 커밋해줘.

4. 현재 브랜치(`minsu`)로 push 해줘.
   - 최초 push면 `git push -u origin minsu`
   - 이미 연결되어 있으면 `git push origin minsu`

5. push 완료 후:
   - GitHub에 PR 만들 수 있는 링크를 알려줘.
   - `minsu` 브랜치에 최신 커밋이 잘 올라갔는지 `git log --oneline -3`으로 확인해줘.
