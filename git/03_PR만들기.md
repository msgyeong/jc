# 🟢 PR 만들기 (minsu → main)

아래 내용을 그대로 실행해줘.

1. 현재 `minsu` 브랜치에서 `origin/minsu`로 push된 최신 커밋을 `git log --oneline -5`로 확인해줘.

2. GitHub에서 Pull Request 만드는 링크를 알려줘.
   - base: `main` ← compare: `minsu`
   - 링크 형식: `https://github.com/msgyeong/jc/compare/main...minsu`

3. PR 설명에 쓸 템플릿을 아래 형식으로 작성해줘:

---
## 변경 요약
- (이번 작업에서 변경한 내용 요약)

## 테스트 방법
- [ ] 로컬에서 `node api/server.js` 실행 후 확인
- [ ] 브라우저에서 `http://localhost:8080` 접속 확인
- [ ] 주요 기능 동작 확인

## 주의사항
- (머지 시 주의할 점, 환경변수 변경 여부 등)
---
